# Learning Support Provider Directory

A React app that lists learning support providers with search, routing, and responsive design.

## Tech Used
- React
- React Router
- Tailwind CSS
- Vite

## Setup
```bash
git clone https://github.com/Roopali-02/habot-directory.git
cd habot-directory
npm install
npm run dev
```
